0-1背包：有个物品，第个物品的体积为，价值为，每个物品至多选一个，求体积和不超过时的最大价值和

回溯三问

当前可以做什么操作？

枚举第个物品选或不选：

  1. 不选，剩余容量不变
  2. 选，剩余容量减少

子问题是什么？

在剩余容量为时，从前个物品中得到的最大价值和

下一个子问题时什么？

进行分类讨论：

  1. 不选时，在剩余容量为时，从前个物品中得到的最大价值和
  2. 选，在剩余容量为时，从前个物品中得到的最大价值和

dfs方法

代码


class Solution{


    public int solution(int[] w,int[] v,int c){
        int n = w.length;
        return dfs(w,v,n-1,c);
    }

    public int dfs(int[] w,int[] v,int i,int c){
        if(i < 0){
            return 0;
        }
        if(c < w[i]){
            return dfs(i-1,c);
        }
        return Math.max(dfs(i-1,c),dfs(i-1,c-w[i])+v[i]);
    }
}

常见变形

  1. 至多装，求方案数/最大价值和

  2. 恰好装，求方案数/最大/最小价值和

  3. 至少装，求方案数/最小价值和
